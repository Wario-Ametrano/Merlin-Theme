# Merlin-Theme
Falkon browser theme

![Dark](https://github.com/Wario-Ametrano/Merlin-Theme/blob/main/Merlin-screen.png "Merlin-Theme")


#### Requirements:

- **Falkon Browser**

## Installation:
Paste the Merlin theme folder into this directory: /usr/share/falkon/themes then in Falkon Browser go to the menu,then to preferences and then to the Appearance option. 

## Set theme:
Select the Merlin Theme in the Appearance Option and apply the theme.
